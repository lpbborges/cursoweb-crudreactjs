## How to install

Use the command bellow to clone the [project](https://github.com/lpbborges/cursoweb-crudreactjs/).

```bash
git clone https://github.com/lpbborges/cursoweb-crudreactjs.git
```

## Back-end

Run
```bash
cd back-end && npm start
```

## Front-end

Run
```bash
cd front-end && npm start
```
